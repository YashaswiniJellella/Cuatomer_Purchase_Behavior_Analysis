# Cuatomer_Purchase_Behavior_Analysis


## Project Description

Customer Purchase Behavior Analysis is a data analytics project developed using Python (Pandas), SQL, and Power BI to analyze customer shopping trends, sales performance, and purchasing behavior.

---

## Problem Statement

Businesses generate large amounts of customer data, but analyzing customer preferences, sales trends, and buying behavior manually is difficult and time-consuming.

---

## Technologies Used

* Python
* Pandas
* SQL
* Power BI

---

## Dashboard Explanation

The dashboard provides interactive visualizations for customer demographics, product categories, revenue analysis, subscription trends, discounts, and sales performance.

---

## Business Insights

The analysis showed that clothing generated the highest revenue, young adults contributed the most purchases, and customers using discounts had higher purchase frequency.

---

## Conclusion

This project helps businesses improve customer engagement, optimize sales performance, enhance marketing strategies, and support data-driven decision-making using analytics and visualization tools.
